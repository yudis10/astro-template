## Getting Started

### 1. Clone this repository

```
git clone [git url] my-website-name
```

### 2. Navigate to the directory

```
cd my-website-name
```

### 3. Install the dependencies

```
npm install
```

### 4. Run Astro

```
npm run dev
```
